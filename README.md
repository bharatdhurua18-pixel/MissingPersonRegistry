# MissingPersonRegistry
The Missing Persons Registry is a web-based application designed to facilitate the reporting, tracking, and dissemination of information about missing persons. Built with Django (Python web framework), the system provides a centralized platform where concerned individuals can report missing persons, view active cases, report sightings, and share information across social media platforms to increase awareness and aid in locating missing individuals.
Project Type: Web Application
Technology Stack: Django 4.2+, Python 3.x, SQLite, Bootstrap 5, HTML5 
Target Users: General public, families of missing persons, law enforcement support
